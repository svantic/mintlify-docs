Svantic is an enterprise AI agent platform that lets you build intelligent agents, connect them to your tools and workflows, and orchestrate them in real time. Agents communicate using the A2A (agent-to-agent) protocol, coordinate through the Svantic Mesh, and are fully observable through distributed tracing and a live dashboard.

## How Svantic works

Svantic has three layers you interact with as a customer:

- **Agents** — Lightweight Node.js services built with `@svantic/sdk`. Each agent exposes capabilities (tools) that the platform can invoke.
- **Mesh** — The agent runtime that routes requests, manages sessions, and orchestrates multi-agent workflows. Your agents connect to the Mesh using `MeshConnector`.
- **Dashboard** — The web UI for monitoring sessions, traces, usage, and configuring your organization.

<Steps>
  <Step title="Install the SDK">
    ```bash
    npm install @svantic/sdk
    ```
  </Step>
  <Step title="Create an agent">
    Define your agent, its capabilities, and expose it over HTTP.
  </Step>
  <Step title="Connect to the Mesh">
    Use `MeshConnector` to register your agent with your Svantic organization.
  </Step>
  <Step title="Start a session and ask">
    Open a session and send natural-language requests — Svantic routes them to the right agents automatically.
  </Step>
</Steps>
